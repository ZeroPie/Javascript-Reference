### Computed Property Names in Objects:

```js
const obj = {
    [action.name]: action.payload
};
```

### Destructuring

```js
state = {
  hot: false,
  cool: true,
  destructured: 'destructured'
}
const { destructured } = this.state

`Element is ${ this.state.hot } and ${destructured}`
```
